# Keyboard Controls

## Module Palette

| Key | Action |
|-----|--------|
| `N` | Open / close the module palette |
| `↑` / `↓` | Navigate the module list |
| `Enter` | Add the highlighted module |
| `Esc` | Close the palette |

---

## Navigating Between Modules

| Key | Action |
|-----|--------|
| `Tab` (on title bar) | Jump to the next module's title bar |
| `Shift+Tab` (on title bar) | Jump to the previous module's title bar |

Focus lands on a module's title bar automatically when it is created.

---

## Module Title Bar

| Key | Action |
|-----|--------|
| `↑` `↓` `←` `→` | Move the module (hold for smooth motion, diagonals supported) |
| `Enter` | Enter the module — focus moves to its first control |
| `Delete` / `Backspace` | Close the module |
| `Esc` | Deselect (blur) the module entirely |

---

## Inside a Module

| Key | Action |
|-----|--------|
| `Tab` / `Shift+Tab` | Cycle through controls (wraps within the module) |
| `Esc` | Return focus to the module title bar |

---

## Patch Cords

| Key | Action |
|-----|--------|
| `Tab` to an output dock, then `Enter` or `Space` | Start drawing a patch cord |
| `Tab` / `Shift+Tab` (while patching) | Cycle through valid input docks only |
| `Enter` or `Space` on an input dock | Complete the connection |
| `Esc` (while patching) | Cancel the patch |
| `Enter` or `Space` on a cord line | Delete that cord |

---

## Preset Bar

| Key | Action |
|-----|--------|
| `Enter` (in name field) | Save preset |
| `Ctrl+S` | Save (focuses the name field first if it is empty) |
| `Ctrl+L` | Open / close the preset dropdown |
| `Ctrl+E` | Export the current patch as a JSON file |
| `Ctrl+O` | Import a preset from a JSON file |
| `Ctrl+Backspace` | Clear all modules and cords (asks for confirmation) |
